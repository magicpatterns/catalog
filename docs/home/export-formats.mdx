We currently export to the following file types: `.js`, `.ts`, `.css`, `.scss`, `.json`. If you're using tailwind, Ww also export to common js to work nicely with your tailwind config.

## Example: Using Mirrorful with CSS Variables

```css
.primary-button {
  background-color: var(--color-primary);
}

.primary-button:hover {
  background-color: var(--color-primary-hover);
}
```

## Example 2: Using Mirrorful with Javascript Constants

```javascript
<button backgroundColor={{ Tokens.primary.base }}>Click here</button>
```
